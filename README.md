## 📌 Project Overview

Dynamic Admin Dashboard UI is a modern and responsive frontend dashboard interface designed for managing users, orders, sales, and business activities.

The dashboard includes a sidebar navigation, top search bar, KPI summary cards, notification/profile section, and a responsive user management table.

---

## ✅ Features

- ✅ Responsive admin dashboard layout
- ✅ Dark sidebar navigation
- ✅ Mobile sidebar toggle
- ✅ Top search bar
- ✅ Notification and profile section
- ✅ KPI summary cards
- ✅ User management table
- ✅ Status badges
- ✅ Edit action column
- ✅ Responsive design using CSS media queries
- ✅ Iconify icon integration
